# Java Modded Export Fixer

A lightweight [Blockbench](https://www.blockbench.net/) plugin that automatically intercepts and modernizes exported Minecraft Java Entity models. It rewrites boilerplate code generated by the default "Java Class" exporter to conform to modern development standards (Minecraft 26.1+).

## Installation

### Manual Installation
1. Download the `java_modded_export_fixer.js` file from this repository.
2. Open Blockbench and navigate to **File > Plugins**.
3. Click the **Load Plugin from File** icon (top right corner of the window).
4. Select the downloaded `.js` file to activate it.
